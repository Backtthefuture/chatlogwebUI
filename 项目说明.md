# 聊天记录查询与AI分析系统

## 项目概述

本项目是一个基于Node.js和Express框架的中文聊天记录查询与AI智能分析系统，提供便捷的聊天数据检索、可视化分析和AI驱动的深度洞察功能。

**当前版本**: v2.5.0

## 安装与配置

### 1. 环境要求
```bash
Node.js >= 18.0.0
npm >= 8.0.0
```

### 2. 安装依赖
```bash
git clone <repository-url>
cd chatlog-web
npm install
```

### 3. 环境配置
创建 `.env` 文件并配置以下参数：
```env
# 基础配置
PORT=3000

# AI模型配置 (通过界面设置)
MODEL_PROVIDER=DeepSeek
DEEPSEEK_API_KEY=your-deepseek-api-key
DEEPSEEK_MODEL=deepseek-chat
GEMINI_API_KEY=your-gemini-api-key  
GEMINI_MODEL=gemini-2.5-pro

# 定时任务配置
ENABLE_SCHEDULED_ANALYSIS=false
SCHEDULED_ANALYSIS_TIME=0 0 8 * * *
```

### 4. 启动服务
```bash
npm start
```

服务启动后访问: http://localhost:3000

## 使用指南

### 模型设置配置
1. 点击页面顶部的"模型设置"按钮
2. 选择AI提供商（DeepSeek或Gemini）
3. 配置对应的模型和API Key
4. 点击"测试连接"验证配置
5. 保存设置

### AI分析使用
1. 选择要分析的群聊
2. 设置时间范围
3. 选择分析类型或输入自定义提示词
4. 点击"执行分析"
5. 查看生成的HTML分析报告

### 定时任务配置
1. 进入定时分析管理区域
2. 配置执行时间（支持Cron表达式）
3. 添加需要分析的群聊和分析项
4. 启用定时任务

## API 接口

### 模型设置相关
- `GET /api/model-settings` - 获取模型设置
- `POST /api/model-settings` - 保存模型设置  
- `POST /api/model-settings/test` - 测试模型连接

### 聊天记录相关
- `GET /api/chatlog` - 获取聊天记录
- `GET /api/contacts` - 获取联系人列表
- `GET /api/chatrooms` - 获取群聊列表

### AI分析相关
- `POST /api/ai-analysis` - 执行AI分析
- `GET /api/analysis-history` - 获取分析历史
- `GET /api/analysis-history/:id` - 获取特定分析记录
